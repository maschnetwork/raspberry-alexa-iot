
# AWS Lamdba Function for Alexa Skills Kit

### Verwendung

Zippe den Inhalt dieses Repositories und lade es in deiner AWS-Lambda Funktion hoch. Dort kannst du im Online-Editor alles weitere editieren. Vergiss nicht zudem deine Lambda-Funktion mit dem Alexa-Skill zu verbinden über "Auslöser hinzufügen" -> "Alexa Skills Kit".
